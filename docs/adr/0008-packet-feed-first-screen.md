# 0008 — Packet feed is the first screen

日期:2026-07-20
状态:已接受

The static site's first screen will serve the external social audience by showing an idea packet feed directly, rather than a project introduction page. This optimizes for classmates opening the link from WeChat and deciding within seconds whether there is something worth reading, forwarding, or discussing; evaluator-oriented reports can live behind a separate entry point.
