# 0021 — 业绩预告文本作为因子原料,全部取自聚源

日期:2026-07-22
状态:已接受(2026-07-22 依实测数据修订)

因子原料层由"行情/财务数值"改为**业绩预告的结构化字段 + 公告正文**,两者**均取自聚源
`JYDB.dbo.LC_PerformanceForecast`**。研究问题的收窄见 ADR-0022。

标的仍是 A 股上市公司 —— alpha 是横截面概念,需要足够多的可比标的、连续可交易价格与公开可比
基本面,只有上市股票同时满足。**换的是因子的原料层,不是资产类别。**

## 数据角色(与 ADR-0011 / 0012 一致,不推翻)

- **聚源 = 唯一数据源。** 价格、前向收益、市值、行业分类、成分股口径,以及预告的结构化字段与正文,
  全部来自聚源。`LC_PerformanceForecast` 同时提供:
  - PIT 锚点 `InfoPublDate`;
  - 结构化字段 `ForcastType` / `ForcastReason` / `EGrowthRateFloor|Ceiling` /
    `EProfitFloor|Ceiling` / `LastProfit`;
  - 分析师一致预期 `NPYOYConsistentForecast`(surprise 的基准);
  - **正文 `ForcastContent`(text 列)**。
- **巨潮爬虫 = 可选补充,当前不使用。** 仅在需要比 `ForcastContent` 更完整的公告原文时才启用。
  这与 `CONTEXT.md` 中 **Research Crawler** 的既有定义一致(supporting tool,not the
  authoritative source for core alpha backtest numbers)。

### 修订说明(原始版本的三处错误)

本 ADR 初稿写于探库之前,以下三条**已被实测推翻**,保留记录以免重蹈:

1. ~~"巨潮为文本原料源"~~ —— 聚源 `ForcastContent` 2019–2021 三年**填充率 100%**,均长
   269–351 字,含"业绩变动原因说明"。文本无需外部来源。
2. ~~"正文需下载并解析 PDF"~~ —— 仅在启用巨潮时才需要;聚源路径下不涉及 PDF。
3. ~~"覆盖率稀疏且非随机,横截面系统性缺失"~~ —— 该结论来自巨潮 `category` 码筛窄造成的假象。
   聚源实测覆盖:2019 年 2921 家、2020 年 3223 家、2021 年 3414 家公司(全市场约 4000 家),
   **约八成**,贵州茅台等白马亦在其中。

## 铁律 1 的边界(必须在论文方法章节写明)

LLM 参与的是**特征工程**,不是回测:

1. 离线批量读 `ForcastContent` → 输出结构化打分 → 按公告 ID 哈希落盘缓存(`data/cache/llm/`);
2. 缓存产物合并为 panel 上的一列普通数值特征;
3. 回测只读这一列,**运行期不发生任何 LLM 调用**。

判据:**回测能否在断网、无 API key 的环境下逐字节复现**。答案必须是能。

## 已知风险与实测约束

- **披露时间粒度不一致。** `InfoPublDate` 部分带精确时刻、部分为 `00:00:00`。统一取保守规则:
  **T 日披露的公告,T+1 交易日起方可使用**,不假设存在盘中时间戳。
- **重复记录多。** 2019-06 至 2021-12 的 39294 行中,按 (code, InfoPublDate, EndDate) 去重后
  剩 19506 行;同一天对多个报告期发预告时须只保留当期,否则会把下一年度指引当作当期业绩。
- **增长率区间缺失约 46%。** 扭亏/首亏类预告没有有意义的增长率(基数为负),这部分标的需要单独口径
  或排除,不能填 0。
- **沪深分布不均。** 深市记录数约为沪市 4 倍(深交所预告强制披露条件更严),行业与板块中性化因此
  更必要。
- **若日后启用巨潮:类别码写错会静默返回全量。** 实测传入伪造的 `category_THIS_IS_FAKE` 与传空
  类别返回条数完全相同(均 729 条),接口不报错。抓取层必须断言返回结果的公告类型并配单测。

## 后果

- 数据契约 `docs/DATA_SCHEMA.md` 需新增第四张表:业绩预告表(代码、披露日、报告期、预告类型、
  幅度区间、一致预期、正文)。
- 无需新增爬虫模块;导出走既有的 `scripts/mssql_tool.py` + `sql/` 路径。
- 交付节奏风险:该方向在五天线末段确立(ADR-0009)。若增量检验不显著,按 ADR-0022,阴性结果
  即为完整结论,不需要退路 —— 但论文必须同时报告结构化字段对照组的完整结果。
