# 0022 — 选题收窄为"文本相对结构化字段的增量检验"

日期:2026-07-22
状态:已接受

研究问题由「智能体能否生成具有经济解释的 A 股 Alpha 因子?」收窄为:

> **当结构化数据已经公开了同一事件的关键数字时,LLM 读文本还能多告诉我们什么?**

检验场:A 股业绩预告(2015–2021)。可证伪形式:LLM 从预告正文中提取的语义特征,相对聚源
`LC_PerformanceForecast` 已有的结构化字段(`EGrowthRateFloor/Ceiling`、`ForcastType`、
`EProfitFloor/Ceiling`),是否具有**增量**的横截面预测力。结构化字段构成的因子是对照组。

## 理由

原问句有三个硬伤:

1. **不可证伪。** 智能体当然"能生成",生成本身不是结果;这个问句永远答"能",是功能声明不是研究问题。
2. **LLM 的位置可被替代。** 在纯数值原料上,LLM 的产出等价于换参数 —— "这跟遗传算法搜表达式有什么区别"
   这个质疑无法回答。
3. **它就是示例题目原话。** 与其他组的作品无法区分。

收窄后的问句同时解决三条:有明确对照组因而增量可量化;LLM 的不可替代性成为被检验的对象而非前提;
问题本身天然可证伪。

## 阴性结果是完整结论

**项目的资产是裁决装置**(确定性回测 / 点时间对齐 / 白名单算子 / 方法依赖图),不是某个因子。
若增量不显著,结论是"市场对该类事件的文本信息已充分定价" —— 这与 McLean & Pontiff (2016) 的
既有发现一致(已在 README 参考文献中),并且**让 ADR-0016 的答辩叙事更强**:系统敢于否决自己提出的
假设,正是它可信的证据。

因此论文与答辩材料不得把"因子有效"写成预期结论,亦不得在增量不显著时隐去该结果。

## 边界

- **仍然是 A 股上市股票。** alpha 是横截面概念,需要足够多的可比标的、连续可交易价格与公开可比基本面,
  只有上市股票同时满足。换的是因子的**原料层**(数值 → 事件文本),不是资产类别。
- **5 个基线因子不作废**,降级为传统量价对照组,主结果表述为"文本因子相对传统因子的增量"。
- 与 ADR-0011(聚源为核心数值源)不冲突:结构化字段与正文均来自聚源,详见 ADR-0021。

## 后果

- `CLAUDE.md`、`README.md`、`CONTEXT.md`(新增 **Increment Test** 术语)、
  `platform/content/showcase/paper.md` 四处的研究问题表述需同步,已一并更新。
- 实证设计必须包含**双因子对照**:仅结构化字段的因子 vs 结构化 + 文本语义的因子,报告增量 IC 与
  增量显著性,而非单独报告后者的 IC。
- 统计口径必须用重叠样本修正后的 t 值(Newey-West),朴素 t 值在持有期重叠下会被高估三到四倍
  (实测 5.93 → 1.60)。
