## 第一章 引言:为什么是这个问题

- **背景**:因子挖掘的两条老路各有硬伤——研究员人肉挖(有经济解释但慢),遗传规划/暴力搜索(快但公式不可读、大概率过拟合)。
- **假设**:LLM 兼具两头优势——像研究员一样"先讲经济逻辑再写公式",又能批量生成。
- **核心主张**:可解释性的含金量在"**事前**"——解释是因子的出生证,不是事后补办。事后让 LLM 解释一个暴力搜索出的公式,它一定能编出漂亮故事,但不可证伪。
- **贡献**:① 一套假说先行的因子生成闭环 ② 一条 LLM 不可触碰的确定性评价链 ③ A 股上的实证漏斗。

## 第二章 相关工作与理论基础

- **alpha 的定义**:组合收益扣掉大盘 β、行业、市值风格三本公共账目后的残余。alpha 是结果,因子是原因,解释是机制。✅
- **横截面转向**:不预测"某只股票的明天"(时间轴信噪比过低),而预测"当日谁比谁强"。→ 评价指标为逐日横截面相关(Rank IC),非时间序列拟合优度。✅
- **检验方法谱系**:Rank IC / ICIR(工业界速查)与 Fama-MacBeth 两步法(学术界正式检验),内核同为"横截面→时间平均"。✅
- **为什么线性**:R² 千分之几的信噪比下,线性是唯一学不了噪声的模型;且用秩替代原始值后,"线性"已放松为"单调"。✅

## 第三章 方法:四步闭环与职责边界

```
生成(LLM) → 校验(规则+LLM) → 回测(纯代码,无 LLM) → 反馈(LLM) → 回到生成
```

- **职责切分**:LLM 管语义空间(带经济先验的假说、自然语言↔表达式翻译、语义查重、定性反馈);代码管数值空间(计算、评价、裁决)。✅
- **受限自主**:控制流硬编码,LLM 在假说空间自由、在评价流程上零自由度。**这是可信度来源,不是能力妥协**——防止既当运动员又当裁判。✅
- **受限表达式引擎**:白名单算子(rank / ts_mean / delay / delta / safe_div …),禁止任意代码执行,既是安全边界也是可解释性保证。✅
- **两层校验**:确定性规则层(字段存在、可解析、是否缺 delay)→ LLM 语义查重层,两层皆过才入回测。✅

## 第四章 数据与防泄漏设计(工程可信度)

- **三张表契约**:`prices`(含 adj_factor 复权)、`fundamentals`(**ann_date 对齐**)、`universe`(历史成分股,防幸存者偏差)。✅
- **pit_merge**:T 日只挂接 `ann_date ≤ T` 的最新财报。反例场景:按 report_period 对齐会让 4-01 的回测用上 4-25 才公告的数据,"未卜先知"躲开暴雷,实盘无法复制。✅ 有单测
- **横截面预处理**:winsorize(压极端值)→ zscore(统一量纲)→ neutralize(剥行业与市值)。中性化是"全市场统一排序"的前提,不是锦上添花。✅
- **一句话立场**:回测的第一美德不是收益高,是诚实还原每个历史时刻的信息集。✅

## 第五章 实证:漏斗与因子卡片(论文主体)

- **核心图表——漏斗**:生成 N 个假说 → 校验过 m 个 → 训练段过 k 个 → 样本外站住 j 个。**负结果也是结果**。⚠️ 待跑真实 LLM
- **对照组**:5 个基线因子(价值/质量/动量/波动/流动性)的 IC 量级。没有对照组的 IC 数字没人信。❌ 待跑
- **健全性检查**:随机/无意义因子的 IC 应≈0。已观察到 mock 因子样本外 IC = -0.0045、ICIR = -0.05,可作为"回测未被 look-ahead 污染"的旁证。✅
- **因子卡片**(每个存活因子一页):假说原文 → 表达式 → IC 曲线 → 分组柱状 → LLM 的经济解释 → 样本外表现。⚠️
- **实验纪律**:每假说最多改 2 轮;调参**选参数平台不选尖峰**;每次改动须讲得出经济理由。✅ 已定

## 第六章 研究纪律:信息单向阀(方法论贡献)

- 训练段随便调、明细全开;测试段封存,最后拆封一次。
- **测试集连"继续/停止"的决策都不许参与**——否则是选择偏差(跑 N 个专挑 OOS 过线的 = 挑运气)。它只做一件事:给结论盖章。
- 需要决策信号则切三段:训练 / 验证 / 测试。本项目两段,答辩时给出三段备案。
- 类比:Kaggle 公开榜 vs 私有榜——无数队伍公开榜前十、私有榜崩盘,就是把公开榜当成了定性指标反复回流。✅

## 第七章 局限与未来工作

- 样本期偏短(当前 2020–2021,训练 1 年 / 测试 1 年),统计显著性受限。⚠️ 建议拉长至 5 年以上
- 自主性受限:属"带反馈循环的多 agent workflow",非完全自主 agent(设计使然,见第三章)。
- 因子拥挤效应未考察:alpha 会随知晓者增多而衰减,长期需持续生产。
- LLM 假说的多样性与重复度未做定量刻画。
