> 给 Codex 的短指令:读 `docs/MANIM_STORYBOARD.md`,按第〇节约定执行,叙事措辞以 `docs/IDEAS.md` 为准。

## 〇、基础约定(硬性,违反任何一条 = 返工)

之前版本"死板"的病根就是:静态卡片堆文字、图一闪而过、公式直接贴结果。以下约定全部反着来:

1. **图就是课,卡片只许收尾。** 每一节的教学发生在动画上:
   - 坐标轴用 `reveal_axes` **一根一根**出现,每根轴讲清"是什么、从哪来、现实含义",讲完停顿;
   - 曲线用 `grow_series` **从左到右长出来**(带描点),让观众"感到"趋势,禁止静态折线直接闪现;
   - 公式用 `derive_formula` **一步一步变换**(每步配一句中文说明),禁止直接贴最终式。
   - 文字卡(`recap_card`)每节最多 1 张、≤2 条,只做"一句话结论"。整页 bullet 文字 = 立即返工。
2. **节奏靠停顿。** 停顿时长用 `read_time(text)` 计算(约 1.6s + 0.16s/字,3–6s 封顶),**禁止把任何停顿压到 1 秒左右**——"全片哗哗闪过"的版本已经被拒过一次。视觉页保持 `page_hold()`(约 5s)。
3. **颜色 = 语义,全片一致。** 本片的固定映射:
   - `BLUE` = 因子 / 信号 / 我们的东西
   - `ORANGE` = 公共收益成分(大盘 β、行业、市值风格)
   - `GREEN` = alpha / 通过 / 合法的信息流向
   - `RED` = 泄漏 / look-ahead / 被拒绝的东西
   - `YELLOW` = 阈值与强调
   任何颜色不许仅作装饰使用。
4. **一支视频一个 Scene。** `ReviewVideo` 从头播到尾,片头(cold_open + 路线图)在前;源码按节拆成 `sections/sNN_*.py`(每节 `NAME` / `COLOR` / `build(s)`),`NAME` 里不许出现 `/`。
5. **中文一律 `Text(..., font=CJK)`,`MathTex` 里只许纯 LaTeX**,中文标签作为独立 `Text` 摆在旁边。`CJK` 常量按 `scripts/check_env.py` 输出设置。
6. **基石节(第 1 节"记账分解")多花 60–90 秒**:用带标签的方块 + 箭头把机制**演**出来,先直觉后公式,不许用一张卡糊弄过去。
7. **版本状态要写清。** 过程版只给团队内部迭代,所有阶段性图示都按"当前理解/待验证"处理;最终展示版必须等完整论文搞定后再生成,所有结论从论文与 `docs/IDEAS.md` 已定措辞中取,不临场发挥。
8. **渲染后必须自查再交付**:`scripts/selfcheck.sh` 抽帧,逐帧检查中文空框、元素重叠、文字裁切。

## 一、分镜(7 节,按序渐进)

片头副标题:**"先讲出道理,再让代码裁决——一个不许 LLM 碰回测的因子工厂"**

片头 `cold_open` 的 4 张 teaser(各配一句短标签,柔和淡入淡出):① 收益分解柱("收益是几笔账") ② 微微上翘的散点云("51.5 : 48.5 的硬币") ③ 四步闭环图("LLM 提想法,代码做裁判") ④ 单向阀箭头("信息只进不回")。

### s01 · 收益记账:alpha 住在哪(★基石节,60–90s)
- `NAME = "收益记账:alpha在哪"`,`COLOR = GREEN`
- helper:自定义方块/箭头机制动画 + `callout` + `derive_formula` + `page_hold`;禁止用 `recap_card` 代替主体。
- 视觉主体:一根"组合收益"总柱,被**逐段拆走**三块横向方块——大盘 β(ORANGE)→ 行业(ORANGE)→ 市值风格(ORANGE),每拆一块配一句"这笔钱指数基金就能买到,不算本事";最后剩一小块 GREEN,callout:"剩余无法归因的部分 = alpha"。
- 镜像箭头:左"公共账目,人人可得(汤)" vs 右"残余,靠假说去抢(肉)"。
- 收尾公式(derive):`R = β·R_m + 行业 + 风格 + α`,一步步长出来。
- 一句话结论:"alpha 是扣完所有公共账目后剩下的钱——是结果,不是原因。"

### s02 · 横截面转向:不预测明天,预测谁比谁强
- `NAME = "横截面转向"`,`COLOR = BLUE`
- helper:`small_axes` + `reveal_axes` + `grow_series` + `callout`;转场用旋转/重排动画表现"转 90°"。
- 视觉 A:轴(x=时间, y=某只股票日收益),`grow_series` 长出一条 RED 噪声线,callout:"单只股票 ≈ 白噪声,时间轴上没戏"。
- 视觉 B(转场要有"旋转 90°"的意象):新轴(x=当日因子分, y=未来收益),散点云微微上翘(BLUE),callout(GREEN):"横截面:排名对了就行"。
- 一句话结论:"时间序列预测'明天',因子投资预测'今天谁比谁强'。"

### s03 · Rank IC:硬币的偏置只在聚合中显形
- `NAME = "Rank IC与硬币偏置"`,`COLOR = BLUE`
- helper:`small_axes` + `reveal_axes` + `grow_series` + `callout` + `derive_formula`;均值线用 YELLOW 虚线。
- 视觉:轴(x=交易日, y=当日 IC),`grow_series` 长出一条围绕 0 上下乱跳、但均值略高于 0 的序列(BLUE);YELLOW 虚线标出均值 0.05,callout:"单日看不出任何东西";再长出第二条累积 IC 曲线(GREEN)稳定上行,callout:"聚合后偏置显形"。
- derive:`ICIR = mean(IC) / std(IC)`,配一句"稳,比高更重要"。
- 一句话结论:"IC=0.05 是一枚 51.5:48.5 的硬币——alpha 是偏置,不是某次抛掷。"

### s04 · 中性化:带阻滤波器
- `NAME = "中性化·带阻滤波"`,`COLOR = ORANGE`
- helper:`small_axes` + `reveal_axes` + 自定义点云/箭头动画 + `callout` + `derive_formula`;不要静态展示前后两张散点图。
- 视觉:散点(x=股票按行业分组排开, y=因子原始分),各行业整团高低不同(ORANGE 团块);动画演"每团减去自己的组均值"——箭头把各团**拉回零轴**,拉平后所有点混在一起(BLUE),callout(GREEN):"现在银行股和芯片股可以同台比了"。
- derive:`因子分 = a·行业 + b·log(市值) + 残差` → 高亮"只留残差"。
- 一句话结论:"中性化 = 滤掉行业频率与市值频率的带阻滤波器(类比傅里叶:投影—分解—取残差)。"

### s05 · 数据管道:pit_merge 与时间穿越
- `NAME = "pit_merge防穿越"`,`COLOR = RED`
- helper:自定义时间轴 + `callout` + `page_hold`;时间点逐个 reveal,红色禁区必须先出现再演暴雷剧情。
- 视觉:一条水平时间轴。YELLOW 标记 `report_period = 3-31`,GREEN 标记 `ann_date = 4-25`;两标记之间的区段涂成 RED 半透明色块,callout(RED):"4-1~4-24 用这份财报 = 时间穿越";一个方块沿时间轴向右滑动(代表回测日 T),滑过 4-25 的瞬间财报数据才"咔哒"挂接上来(变 GREEN)。
- 加一个暴雷剧情 callout:"4-25 公告暴雷——按 report_period 对齐的回测会'未卜先知'地提前躲开"。
- 一句话结论:"回测的第一美德不是收益高,是诚实还原每个时刻的信息集:T 日只用 ann_date ≤ T。"

### s06 · 四步闭环:LLM 提想法,代码做裁判
- `NAME = "四步闭环"`,`COLOR = PURPLE`
- helper:自定义节点环 + `callout` + `page_hold`;节点必须按顺序点亮,回测节点带锁并停顿。
- 视觉:四个节点顺时针依次点亮成环——生成(BLUE,LLM 图标)→ 校验(BLUE+GREEN 双层)→ **回测(GREEN,带锁图标,旁注"无 LLM")** → 反馈(BLUE)→ 箭头回到生成。回测节点点亮时全环停顿,callout(YELLOW):"生杀大权在不含 LLM 的代码手里"。
- 一句话结论:"LLM 管语义空间(假说/翻译/反馈),代码管数值空间(计算/裁决)——不许既当运动员又当裁判。"

### s07 · 信息单向阀:封存不是仪式,是物理隔离
- `NAME = "信息单向阀"`,`COLOR = GREEN`
- helper:自定义时间轴/箭头 + `callout` + 漏斗动画 + `page_hold`;反向箭头必须被 RED 大叉打断。
- 视觉:时间轴劈成两段——训练段(BLUE,`train_end` 之前)与测试段(灰色+锁图标)。GREEN 粗箭头从训练指向测试("带着所学去赴考");一根反向箭头刚伸出来就被打上 RED 大叉,callout(RED):"哪怕只瞟一眼,泄漏就已完成"。
- 收尾:漏斗动画——"N 个假说 → 训练段过 m 个 → 样本外站住 k 个"三层漏斗逐层收窄(这是答辩主图,多停 3 秒)。
- 一句话结论:"限制信息的时间范围,不限制段内粒度;测试集只做一件事——给结论盖章。"
